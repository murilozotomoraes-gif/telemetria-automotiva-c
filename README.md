# telemetria-automotiva-c
Repositório para registrar minha evolução em lógica de programação e linguagem C, focado em sistemas de telemetria.

Este projeto faz parte dos meus estudos em **Engenharia de Computação na UTFPR**. O objetivo é aplicar conceitos de lógica em C simulando a leitura de sensores de um veículo.

### Status Atual: Fundamentos
Nesta versão, o código utiliza:
- Estruturas de repetição (`while`, `for`).
- Condicionais (`if/else`).
- Vetores para histórico de dados.
- Funções para modularização.

### Como rodar
1. Tenha um compilador de C instalado (GCC).
2. Compile com: `gcc main.c -o telemetria`
3. Execute: `./telemetria` (ou `telemetria.exe` no Windows).
